## 💡 快速使用指南

### 在新组件中使用俄语:

```vue
<template>
  <div>
    <!-- 直接使用 -->
    <h1>{{ $t('scenic.title') }}</h1>
    <el-button>{{ $t('common.save') }}</el-button>
    
    <!-- 带参数 -->
    <span>{{ $t('form.pleaseEnter', { field: $t('user.username') }) }}</span>
  </div>
</template>

<script setup>
import { useI18n } from 'vue-i18n'
const { t } = useI18n()

// 在 JavaScript 中使用
function handleSave() {
  ElMessage.success(t('message.saveSuccess'))
}

// 在验证规则中使用
const rules = {
  username: [
    { required: true, message: t('form.required', { field: t('user.username') }), trigger: 'blur' }
  ]
}
</script>
```

### 常用翻译键速查:

```javascript
// 操作
$t('common.save')      // Сохранить
$t('common.cancel')    // Отменить
$t('common.confirm')   // Подтвердить
$t('common.delete')    // Удалить
$t('common.edit')      // Редактировать
$t('common.add')       // Добавить
$t('common.search')    // Поиск
$t('common.reset')     // Сбросить

// 消息
$t('message.success')  // Операция выполнена успешно
$t('message.failed')   // Операция не выполнена
$t('message.saveSuccess')  // Сохранено успешно
$t('message.deleteSuccess') // Удалено успешно

// 用户
$t('user.username')    // Имя пользователя
$t('user.password')    // Пароль
$t('user.email')       // Email
$t('user.phone')       // Телефон
$t('user.profile')     // Профиль
$t('user.logout')      // Выйти

// 表单
$t('form.pleaseEnter', { field: '字段' })  // Пожалуйста, введите...
$t('form.pleaseSelect', { field: '字段' }) // Пожалуйста, выберите...
$t('form.required', { field: '字段' })     // ... обязательно
```

## 📞 技术支持

### 常见问题:

**Q: 翻译不显示怎么办?**
A: 检查是否导入了 `useI18n` 并正确使用 `$t()` 或 `t()`

**Q: Element Plus 还是英文?**
A: 检查 `main.js` 中是否正确导入并配置了 `ruLocale`

**Q: 如何添加新翻译?**
A: 在 `web/src/i18n/locales/ru.js` 中添加新的键值对

**Q: 后端消息如何本地化?**
