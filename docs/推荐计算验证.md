# 协同过滤推荐计算验证

## 手动计算

### 用户相似度

共同景点：华山(21)、呼伦贝尔(26)

余弦相似度计算：
- 点积 = 5.0×5.0 + 3.0×5.0 = 25 + 15 = 40
- 用户2模 = √(25+64+9+9+64) = √171 ≈ 13.08
- 用户3模 = √(25+25) = √50 ≈ 7.07
- **相似度 = 40 / (13.08 × 7.07) ≈ 0.432**

### 为用户2推荐

用户3喜欢但用户2未交互的景点：
- **无**（用户2已经有呼伦贝尔26了）

结论：用户2没有推荐（或返回热门景点）

### 为用户3推荐

用户2喜欢但用户3未交互的景点：
1. **厦门鼓浪屿(25)**: 0.432 × 8.0 = 3.456分
2. **少林寺(29)**: 0.432 × 8.0 = 3.456分  
3. **纳木错(28)**: 0.432 × 3.0 = 1.296分

结论：用户3应该被推荐这3个景点

## 预期结果

- 用户2：空推荐或热门景点
- 用户3：厦门鼓浪屿、少林寺、纳木错
