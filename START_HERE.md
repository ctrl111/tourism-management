```
├── ScenicDetails.vue       ← Обновить
├── TravelNote.vue          ← Обновить
└── TravelDetails.vue       ← Обновить
```

#### Как обновить компонент:

1. Откройте файл (например, `Profile.vue`)
2. Добавьте в `<script setup>`:
```vue
import { useI18n } from 'vue-i18n'
const { t } = useI18n()
```

3. Замените китайский текст:
```vue
<!-- Было -->
<el-button>保存</el-button>

<!-- Стало -->
<el-button>{{ $t('common.save') }}</el-button>
```

4. В JavaScript коде:
```javascript
// Было
ElMessage.success('操作成功')

// Стало
ElMessage.success(t('message.success'))
```

1. Откройте IntelliJ IDEA / VS Code

3. Выберите область: `src/main/java`

## 🆘 ПОМОЩЬ

### Проблема: Переводы не отображаются

**Решение:** Убедитесь, что:
1. i18n импортирован в `main.js`
2. Используете `$t()` в шаблонах
3. Используете `t()` в скриптах после `const { t } = useI18n()`

### Проблема: Element Plus на английском

**Решение:** Проверьте `web/src/main.js`:
```javascript
import ruLocale from 'element-plus/dist/locale/ru.mjs'

app.use(ElementPlus, {
  locale: ruLocale
})
```

### Проблема: Backend возвращает китайские сообщения

**Решение:** Выполните замены из `MANUAL_JAVA_LOCALIZATION.md`
