### Шаг 2: Запустите frontend


## 🔧 Полная локализация

### Frontend (Vue компоненты)

Используйте шаблон:

```vue
<template>
  <div>
    <h1>{{ $t('scenic.title') }}</h1>
    <el-button>{{ $t('common.save') }}</el-button>
  </div>
</template>

<script setup>
import { useI18n } from 'vue-i18n'
const { t } = useI18n()

// В коде
ElMessage.success(t('message.success'))
</script>
```

### Backend (Java)

Используйте Find & Replace в IDE:

```
Найти: "操作成功"
```

## 📞 Поддержка

### Если что-то не работает:

1. **Переводы не отображаются:**
   - Проверьте, что i18n импортирован в main.js
   - Убедитесь, что используете `$t()` в шаблонах

2. **Element Plus на английском:**
   - Проверьте импорт русской локали в main.js

3. **Backend возвращает китайские сообщения:**
   - Выполните замены из MANUAL_JAVA_LOCALIZATION.md
