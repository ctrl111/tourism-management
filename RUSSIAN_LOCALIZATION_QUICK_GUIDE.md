## 🔧 如何查看俄语界面

### 如果看到英文技术名称（如"menu.touristManage"）

这是浏览器缓存问题，请按以下步骤操作：

1. **强制刷新页面**
   - Windows: `Ctrl + Shift + R`
   - Mac: `Cmd + Shift + R`

2. **或者清除浏览器缓存**
   - 打开浏览器设置
   - 清除"缓存的图片和文件"
   - 刷新页面

3. **开发环境重启**
   ```bash
   cd web
   npm run dev
   ```
