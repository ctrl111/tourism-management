# Ручная локализация Java файлов

Если Python не установлен, используйте эту инструкцию для ручной замены текстов в Java файлах.

## Метод 1: Использование Find & Replace в IDE

### В IntelliJ IDEA / VS Code:

1. Откройте проект
2. Нажмите `Ctrl+Shift+R` (Replace in Files)
3. Выберите область поиска: `src/main/java`
4. Включите опцию "Match case" (учитывать регистр)
5. Выполните замены по списку ниже

## Метод 2: Использование скрипта PowerShell (если есть проблемы с кодировкой)

Создайте файл `replace.ps1` с содержимым:

```powershell
$replacements = @{
    '"操作成功"' = '"Операция выполнена успешно"'
    '"登录成功"' = '"Вход выполнен успешно"'
    # ... добавьте остальные замены
}

Get-ChildItem -Path "src\main\java" -Filter "*.java" -Recurse | ForEach-Object {
    $content = Get-Content $_.FullName -Raw -Encoding UTF8
    $modified = $false
    
    foreach ($key in $replacements.Keys) {
        if ($content -match [regex]::Escape($key)) {
            $content = $content -replace [regex]::Escape($key), $replacements[$key]
            $modified = $true
        }
    }
    
    if ($modified) {
        Set-Content -Path $_.FullName -Value $content -Encoding UTF8 -NoNewline
        Write-Host "Updated: $($_.Name)"
    }
}
```

Запустите:
```powershell
.\replace.ps1
```

## Проверка результатов

После замены проверьте несколько файлов:

1. `src/main/java/com/project/platform/controller/UserController.java`
2. `src/main/java/com/project/platform/service/impl/UserServiceImpl.java`
3. `src/main/java/com/project/platform/dto/LoginDTO.java`

Убедитесь, что китайские тексты заменены на русские.

3. Протестируйте API endpoints
