# 清除浏览器缓存以查看俄语翻译

## 问题
后台管理菜单显示的是英文技术名称（如 "menu.touristManage"）而不是俄语翻译。

## 原因
这是因为浏览器缓存了旧版本的JavaScript文件。所有俄语翻译已经正确配置，只需要清除缓存即可。

## 解决方案

### 方法1：强制刷新（推荐）
1. 打开浏览器中的后台管理页面
2. 按以下快捷键强制刷新：
   - **Windows**: `Ctrl + Shift + R` 或 `Ctrl + F5`
   - **Mac**: `Cmd + Shift + R`
   - **Linux**: `Ctrl + Shift + R`

### 方法2：清除浏览器缓存
1. 打开浏览器设置
2. 找到"隐私和安全"或"清除浏览数据"
3. 选择"缓存的图片和文件"
4. 点击"清除数据"
5. 刷新页面

### 方法3：使用开发者工具
1. 按 `F12` 打开开发者工具
2. 右键点击浏览器刷新按钮
3. 选择"清空缓存并硬性重新加载"

### 方法4：重新构建前端（开发环境）
如果您在开发环境中，可以重新构建前端：

```bash
cd web
npm run dev
```

或者构建生产版本：

```bash
cd web
npm run build
```

## 技术说明

所有翻译已正确配置在以下文件中：
- `web/src/i18n/locales/ru.js` - 俄语翻译文件
- `web/src/i18n/index.js` - i18n配置
- `web/src/main.js` - i18n集成到Vue应用

菜单组件使用Vue I18n的 `$t()` 函数：
```vue
<span>{{ $t('menu.systemName') }}</span>
```

这会自动从 `ru.js` 中查找对应的俄语翻译。

## 如果问题仍然存在

1. 检查浏览器控制台是否有JavaScript错误
2. 确认 `web/src/i18n/locales/ru.js` 文件没有语法错误
3. 确认前端开发服务器正在运行
4. 尝试在隐私/无痕模式下打开页面

## 答辩准备

清除缓存后，整个后台管理系统将完全显示为俄语，包括：
- 所有菜单项
- 所有按钮和标签
- 所有错误消息
- 所有表单验证提示

这将为您的俄语答辩提供完美的演示效果！
