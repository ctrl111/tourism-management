## 🔧 Структура таблиц

**Важно:** Структура таблиц НЕ требует изменений!

- Кодировка `utf8mb4` полностью поддерживает кириллицу
- Все поля COMMENT остаются на китайском (это только метаданные)
- Данные в таблицах полностью на русском языке

## 📝 Как применить изменения

### Вариант 1: Полная переинициализация (рекомендуется)

```bash
# 1. Удалить существующую базу данных
mysql -u root -p -e "DROP DATABASE IF EXISTS tourism_db;"

# 2. Создать новую базу данных
mysql -u root -p -e "CREATE DATABASE tourism_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 3. Импортировать все SQL-файлы по порядку
mysql -u root -p tourism_db < "sql (1)/tourism_db_user.sql"
mysql -u root -p tourism_db < "sql (1)/tourism_db_scenic_category.sql"
mysql -u root -p tourism_db < "sql (1)/tourism_db_scenic_info.sql"
mysql -u root -p tourism_db < "sql (1)/tourism_db_order_info.sql"
mysql -u root -p tourism_db < "sql (1)/tourism_db_notice.sql"
mysql -u root -p tourism_db < "sql (1)/tourism_db_travel_note.sql"
mysql -u root -p tourism_db < "sql (1)/tourism_db_comment_info.sql"
mysql -u root -p tourism_db < "sql (1)/tourism_db_favorite.sql"
```
