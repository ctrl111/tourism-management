## 🎯 俄语本地化特点

### 符合俄罗斯应用习惯的翻译

1. **正式但友好的语气**
   - 使用 "Вы" (正式的"您") 而不是 "ты" (非正式的"你")
   - 例如: "Войдите в систему" (请登录系统)

2. **简洁明了的表达**
   - "Операция выполнена успешно" (操作成功完成)
   - "Данные обновлены" (数据已更新)

3. **符合俄语语法**
   - 正确的格变: "Изменение пароля" (密码的更改)
   - 正确的性别一致: "Пароль изменён" (密码被更改了)

4. **本地化的术语**
   - "Баланс счёта" (账户余额) - 使用俄罗斯银行术语
   - "Пополнить счёт" (充值账户) - 符合俄罗斯支付习惯

## 💡 快速本地化模板

### Vue 组件本地化模板:

```vue
<script setup>
import { useI18n } from 'vue-i18n'
const { t } = useI18n()
</script>

<template>
  <!-- 替换硬编码文本 -->
  <h1>{{ $t('category.key') }}</h1>
  <el-button>{{ $t('common.save') }}</el-button>
  
  <!-- 在 JavaScript 中 -->
  <el-button @click="handleClick">
    {{ $t('common.confirm') }}
  </el-button>
</template>

<script>
function handleClick() {
  ElMessage.success(t('message.success'))
}
</script>
```

### 常用翻译键:

```javascript
// 通用操作
$t('common.save')      // 保存
$t('common.cancel')    // 取消
$t('common.confirm')   // 确认
$t('common.delete')    // 删除
$t('common.edit')      // 编辑

// 消息
$t('message.success')  // 操作成功
$t('message.failed')   // 操作失败

// 表单
$t('form.pleaseEnter', { field: '字段名' })  // 请输入...
$t('form.pleaseSelect', { field: '字段名' }) // 请选择...
```
