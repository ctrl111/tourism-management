## 🎯 俄语本地化质量标准

### 符合俄罗斯应用习惯 ⭐⭐⭐⭐⭐

#### 1. 语言风格 ✅
- 使用正式的 "Вы" (您) 称呼
- 礼貌用语: "Пожалуйста" (请), "Спасибо" (谢谢)
- 专业、友好的语气

#### 2. 语法正确性 ✅
- 正确的格变 (падежи)
- 正确的性别一致 (род)
- 正确的数的一致 (число)
- 符合俄语语法规则

#### 3. 本地化术语 ✅
```
中文 → 俄语 (符合本地习惯)
─────────────────────────────
账户余额 → Баланс счёта (银行术语)
充值 → Пополнить счёт (支付习惯)
订单 → Заказ (标准术语)
景点 → Достопримечательность (旅游术语)
游记 → Заметка о путешествии (地道表达)
收藏 → Избранное (常用词汇)
```

#### 4. UI 文本规范 ✅
```
按钮文本 (动词不定式):
- Сохранить (保存)
- Отменить (取消)
- Войти (登录)
- Выйти (退出)

标签文本 (名词):
- Имя пользователя (用户名)
- Электронная почта (电子邮箱)
- Телефон (电话)

提示信息 (完整句子):
- Пожалуйста, введите имя пользователя
- Пароль должен содержать не менее 6 символов
- Операция выполнена успешно
```

### 在新组件中使用俄语

#### 模板示例:
```vue
<template>
  <div>
    <!-- 使用翻译 -->
    <h1>{{ $t('scenic.title') }}</h1>
    <el-button>{{ $t('common.save') }}</el-button>
    
    <!-- 带参数的翻译 -->
    <span>{{ $t('form.pleaseEnter', { field: $t('user.username') }) }}</span>
  </div>
</template>

<script setup>
import { useI18n } from 'vue-i18n'
const { t } = useI18n()

// JavaScript 中使用
function handleSave() {
  ElMessage.success(t('message.saveSuccess'))
}
</script>
```

### 常用翻译键速查表

#### 操作类:
```javascript
$t('common.save')      // Сохранить (保存)
$t('common.cancel')    // Отменить (取消)
$t('common.confirm')   // Подтвердить (确认)
$t('common.delete')    // Удалить (删除)
$t('common.edit')      // Редактировать (编辑)
$t('common.add')       // Добавить (添加)
```

#### 消息类:
```javascript
$t('message.success')       // Операция выполнена успешно
$t('message.failed')        // Операция не выполнена
$t('message.saveSuccess')   // Сохранено успешно
$t('message.deleteSuccess') // Удалено успешно
```
