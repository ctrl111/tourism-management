## 使用说明

### 登录流程
1. 用户访问登录页面 `/login`
2. 输入用户名、密码、选择用户类型
3. 输入验证码（验证码区分大小写）
4. 可选择"记住我"保存用户名
5. 点击登录或按回车键提交
6. 登录成功后根据用户类型或 redirect 参数跳转
   - 管理员 → `/admin`
   - 普通用户 → `/front/index`
   - 有 redirect → 跳转到 redirect 指定的页面

### 路由保护
- **白名单路由**：无需登录即可访问
  - `/login` - 登录页
  - `/register` - 注册页
  - `/retrievePassword` - 找回密码
  - `/front/index` - 前台首页

- **需要登录的路由**：
  - `/admin/*` - 所有管理员路由（仅管理员可访问）
  - `/front/personalCenter` - 个人中心
  - `/front/orders` - 我的订单
  - `/front/favorite` - 我的收藏
  - `/front/viewHistory` - 浏览历史

- **其他前端路由**：部分开放，可以游客身份浏览

### 状态持久化
- Token 保存在 localStorage
- 用户信息缓存在 localStorage
- 记住我选项保存用户名
- 页面刷新后自动恢复登录状态
