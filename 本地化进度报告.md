## 🎯 符合俄罗斯习惯的翻译示例

### 1. 礼貌用语
```
中文 → 俄语
请输入 → Пожалуйста, введите
请选择 → Пожалуйста, выберите
谢谢 → Спасибо
```

### 2. 专业术语
```
账户余额 → Баланс счёта (银行术语)
充值 → Пополнить счёт (支付习惯)
订单 → Заказ (标准术语)
景点 → Достопримечательность (旅游术语)
```

### 3. 按钮文本
```
保存 → Сохранить (动词不定式)
取消 → Отменить
确认 → Подтвердить
删除 → Удалить
```

### 4. 系统消息
```
操作成功 → Операция выполнена успешно
操作失败 → Ошибка выполнения операции
确定要删除吗？ → Вы уверены, что хотите удалить?
```
