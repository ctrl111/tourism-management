## ⚠️ Если что-то не работает

### Стили не применяются?
```bash
# Очистите кэш и перезапустите
cd web
rm -rf node_modules/.vite
npm run dev
```

### Тексты все еще обрезаются?
Откройте `web/src/styles/russian-ui-fixes.css` и увеличьте:
```css
.el-form-item__label {
  min-width: 160px !important; /* было 120px */
}
```

### Карточки разной высоты?
Откройте `web/src/styles/front.css` и измените:
```css
.scenic-title {
  min-height: 50px; /* было 44px */
}
```
